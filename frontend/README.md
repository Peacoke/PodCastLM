## Installation

To set up the project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone git@github.com:YOYZHANG/PodCastLM.git
   cd PodCastLM/frontend
   ```

2. **install dependencies:**
   ```bash
   pnpm install
   ```

## Usage

```sh
pnpm run dev
```
